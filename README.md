# 功能介绍

- 支持夜间模式
- 搜索功能
- 自定义色彩
- 支持谷歌统计
- 支持 Valine 评论
- 响应式布局
- 自定义字体
- …

# 安装

将 RoFty，放入**文档/Gridea/themes 目录**，重启应用即可选择使用。
